Carcassonne (Base A–X)

Tile

Rotation: 0°
Wheel over board rotates tile.
Arrow keys move board (hover board).
Hover empty cell shows ghost.
Place a tile by clicking an empty cell.
Right-click a tile to remove it (and attached meeples).
Click a placed tile to show meeple markers.

Meeples

Click a placed tile to show feature markers, then click a marker to place a meeple for the currently selected player.

Scoring

“Score” = points already awarded (completed features).
“End-if-now” = additional points if the game ended now (incomplete claimed features + farms).
“If-complete-now” = value if currently claimed roads/cities/cloisters were completed with their current extent.

Actions

Placement rules implemented

  • Edge matching (road/city/field primary).
  • Road/City completion: all open edge-ports connected.
  • Cloister completion: 8 surrounding tiles.
  • Majority scoring; ties split full points.
  • Farms (end scoring): 3 per completed adjacent city (base rules).
